About Spirographs:

Spirographs are mathematical curves created by rolling a circle around another circle. Adjust the parameters to create different patterns:

  • Outer Radius (R): Size of fixed circle
  • Inner Radius (r): Size of rolling circle
  • Distance (d): Distance from center to drawing point
  • Resolution: Higher values create smoother curves

Try different combinations to discover unique patterns!
